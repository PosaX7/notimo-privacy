# Politique de confidentialité - NoTiMo

Dernière mise à jour : Avril 2026

## 1. Introduction

L'application NoTiMo respecte votre vie privée. Cette politique explique quelles informations sont collectées et comment elles sont utilisées.

## 2. Données collectées

NoTiMo ne nécessite pas la création de compte utilisateur.

Les données suivantes peuvent être stockées localement sur votre appareil :
- Informations financières saisies par l’utilisateur (revenus, dépenses, budgets)
- Données d’utilisation de l’application

Ces données restent sur votre appareil et ne sont pas transmises à des serveurs externes.

## 3. Utilisation des données

Les données sont utilisées uniquement pour :
- Fournir les fonctionnalités de l’application
- Améliorer l’expérience utilisateur

## 4. Partage des données

NoTiMo ne vend, n’échange et ne partage aucune donnée personnelle avec des tiers.

## 5. Sécurité

Les données sont stockées localement sur votre appareil. Vous êtes responsable de la sécurité de votre appareil.

## 6. Suppression des données

Vous pouvez supprimer vos données à tout moment en désinstallant l’application.
